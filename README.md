## 📖 Project Overview

This project is a front-end clone of the Amazon homepage created using HTML5 and CSS3. It recreates the layout, navigation bar, hero banner, product sections, and footer while focusing on responsive design and clean UI development.

## ✨ Features

- Responsive navigation bar
- Amazon-style logo and search bar
- Hero banner section
- Product category cards
- Shopping sections with images
- Responsive footer
- Clean and organized UI
- Built using only HTML and CSS

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/riyamishra0226/AMAZON-PROJECT.git
```

2. Open the project folder.

3. Run the project by opening `index.html` in your browser or using the **Live Server** extension in VS Code.
